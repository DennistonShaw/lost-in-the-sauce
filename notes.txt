Quick commands
pwd = print working directory *means show the current folder I’m in right now
cd = change directory. Type cd and the name of directory
cd .. = going back one directory
ls (list storage) = allow you to view all the files inside the current directory
Clear = clears out the screen

Important to have a system to search for things

How to create a file using terminal:
touch and name ie “touch lizzo.txt” to create a file
rm and name to remove the file “rm touch lizzo.txt”
